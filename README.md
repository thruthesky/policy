## 형식

파일 형식 지정

## 개인정보 보호 정책, Privacy Policy

- `[도메인]-privacy.txt`


위에서 `도메인` 에는 `withcenter`, `philgo`, `sonub` 와 같이 extension 은 빼고 입력. 앱 이름을 입력해도 된다.

## 가입약관 - Terms and Use

- `[도메인]-terms.txt`

위에서 `도메인` 에는 `withcenter`, `philgo`, `sonub` 와 같이 extension 은 빼고 입력. 앱 이름을 입력해도 된다.
